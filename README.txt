# Ассистент Sales-менеджера (AmoCRM + Голос)

Этот агент помогает менеджерам по продажам работать с AmoCRM через Telegram-бота, поддерживая голосовой ввод.

## Основные возможности:
1. **Голосовой ввод**: Автоматическое распознавание речи и передача текста ИИ.
2. **AmoCRM**: Поиск контактов и создание сделок (через LLM-инструменты).
3. **Умный помощник**: GPT-4o анализирует запрос и выбирает нужное действие.

## Как запустить:
1. Импортируйте папку проекта в Agent Platform.
2. В разделе **Integrations** настройте:
   - `telegram`: Токен вашего бота.
   - `amocrm_oauth`: Подключите аккаунт AmoCRM.
   - `openai_whisper`: Ключ OpenAI для распознавания речи.
3. Опубликуйте проект.

## Структура:
- `nodes.json`: Логика диалога (Триггер -> Проверка голоса -> ASR -> Агент -> Ответ).
- `userFunctions.json`: Код для API запросов к AmoCRM.
- `metadata.json`: Базовые настройки проекта.
